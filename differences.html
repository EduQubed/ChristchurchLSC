Differences between HTML4 & HTML5

HTML4

HTML4.01 and older used divs (dividers) to divide the page into the sections you require.

For example

<div id="nav"> <div class="header"> <div id="footer">

HTML5

HTML5 is supported by all modern browsers, however for the older browsers you can use a shiv to teach older browsers how to render elements correctly.

divs can still be used in HTML5, however the newer semantic elements are used to help with SEO (Search Engine Optimisation) and also to aid screen readers for accesibility

For example

<nav> <header> <footer>

The above example is now used in place of the HTML4.01 and below version at the top of this page.

Here are the new HTML5 semantic elements which can be used:

<article> <aside> <details> <figcaption> <figure> <footer> <header> <main> <mark> <nav> <section> <summary> <time>
